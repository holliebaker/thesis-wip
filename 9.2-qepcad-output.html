Example 9.2 Let \(n = 3\) and consider the unit sphere \[ C := \{ (x,y,z) \in \mathbb{R}^3 \mid (x,y) \in C', z = \varphi(x,y) \} \] where \[ C' := \{ (x,y) \in \mathbb{R}\mid -1 < x < 1, y > 0, x + y < 1 \} \] and \[ \varphi(x,y) = x^2 + y^2. \]

  • Quasi-affine:

    \(C\) is already quasi-affine, since \(\varphi\) is strictly incleasing in both \(x\) and \(y\) on \(C'\).

  • monotone

    \(C\) is a \((0,0,1)\)-cell, and its top and bottom are the graphs of \(\varphi\vert_{C'_B}\) and \[\varphi\vert_{C'_T}$, where $C'_B := \{ (x,y) \in \mathbb{R}^2 \mid 0 < x < 1, y = 0 \}$ and $C'_T := \{ (x,y) \in \mathbb{R}^2 \mid 0 < x < 1, x + y < 0 \}$ Following equation \@ref(eq:jacobi-det), compute \]

    \[\begin{pmatrix}1 & 1\\ 2x & 2y \end{pmatrix}\]

    =2y-2y, \[ then solve \] { x^2 + y^2 - z, 2x + 2y } $$ by computing \({\operatorname{proj}_{{\operatorname{span} \{x_1\}}}}\) to obtain \(2x - 1 = 0\). Observe that this is the local minimam of \(\varphi\vert_{C'_T}\) where \(C'_T = \{ (x,y) \in \mathbb{R}^2 \mid -1 < x < 1, x + y = 1 \}\).

    Hence, \(\mathcal{R}_1 = \{ (1/2) \}\), and \(C\) is split into two monotone \((1,1,0)\)-cells and one monotone \((0,1,0)\)-cell.

  • Frontier Condition:

    No blow-up points, already satisfied.

:::
